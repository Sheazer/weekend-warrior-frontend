🚀 Activity Finder Frontend

Современное frontend-приложение для поиска, создания и отображения активностей на карте.

Проект разработан с использованием React + Vite и взаимодействует с backend API на Go.

📌 Возможности проекта

✅ Просмотр списка активностей
✅ Создание событий
✅ Отображение событий на карте
✅ Поиск активностей
✅ Фильтрация событий
✅ Авторизация пользователей
✅ Работа с JWT токенами
✅ Выбор локации через карту
✅ Popup информация о событиях

⚙️ Что нужно установить

Перед запуском проекта должны быть установлены:

1️⃣ Node.js

Скачать:

https://nodejs.org/

Проверка установки:

node -v
npm -v
2️⃣ JavaScript

JavaScript отдельно скачивать не нужно.
Он уже встроен в Node.js и браузер.

📦 Установка проекта
Клонирование репозитория
git clone <repo_url>
Переход в папку проекта
cd frontend
Установка зависимостей
npm install
▶️ Запуск проекта
Development режим
npm run dev

После запуска Vite покажет локальный адрес:

http://localhost:5173
📚 Используемые библиотеки
React Router DOM

Используется для маршрутизации страниц.

Установка:

npm install react-router-dom
React Leaflet + Leaflet

Используется для отображения карты и маркеров.

Установка:

npm install react-leaflet leaflet
🗺️ Карта и геолокация

На карте отображаются активности пользователей.

Каждая активность содержит координаты:

{
  "latitude": 42.87,
  "longitude": 74.60
}

Используется:

OpenStreetMap
Leaflet
React Leaflet
🔐 Авторизация

В проекте используется JWT авторизация.

После входа:

token сохраняется в localStorage
пользователь получает доступ к созданию событий
открывается профиль пользователя
