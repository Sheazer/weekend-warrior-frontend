# 🚀 Activity Finder Frontend

Современное и технологичное frontend-приложение для поиска, создания и отображения различных активностей на интерактивной карте. 

Проект разработан с использованием связки **React + Vite** и оптимизирован для полноценного взаимодействия с backend API на **Go**.

---

## ✨ Возможности проекта

* **📍 Интерактивная карта** — отображение всех событий вокруг вас в реальном времени.
* **🔍 Умный поиск** — удобная фильтрация активностей по ключевым словам и категориям.
* **🔐 Безопасность** — полноценная авторизация пользователей с использованием JWT-токенов.
* **🛠️ Конструктор событий** — создание собственных активностей с выбором точной локации кликом по карте.
* **💬 Информационные Popups** — быстрый просмотр деталей карточки события без перезагрузки страницы.

## 🗺️ Карта и геолокация

Каждое созданное событие сохраняет свои точные географические координаты. Пример структуры объекта локации:

```json
{
  "latitude": 42.87,
  "longitude": 74.60
}
```
Визуализация данных обеспечивается за счет интеграции карт **OpenStreetMap** через обертку **React Leaflet**.

---

## 🔐 Авторизация и безопасность

Безопасность пользовательских данных реализована на базе **JWT (JSON Web Tokens)**:

* После успешного входа валидный `token` сохраняется в `localStorage`.
* Наличие токена открывает доступ к защищенным роутам (создание и редактирование событий, просмотр личного профиля).
* Токен автоматически прикрепляется к заголовкам API-запросов.

## 🔍 Функционал поиска и создания

### Поиск активностей
Пользователям доступен гибкий поиск, который фильтрует массив данных по:
* Названию события
* Текстовому описанию
* Выбранной категории (спорт, отдых, учеба и т.д.)

### 📍 Создание активности
При генерации новой точки на карте форма требует заполнения следующих параметров:
* Название и подробное описание
* Категория
* Дата и время проведения
* Лимит количества участников
* Координаты (автоматически подставляются при клике на карту)
